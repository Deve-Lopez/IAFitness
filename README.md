IAFitness es una aplicación de fitness y bienestar que integra inteligencia artificial para ofrecer rutinas personalizadas según el estado físico, motivación y objetivos del usuario. El proyecto está actualmente en desarrollo.


Descripción
El objetivo de IAFitness es combinar tecnología y entrenamiento personalizado, generando rutinas adaptadas a cada usuario a través de cuestionarios diarios y análisis de datos. La aplicación está diseñada para dispositivos móviles, ofreciendo una experiencia intuitiva y eficiente.


Características (planeadas):
Registro e inicio de sesión de usuarios.
Cuestionario diario sobre motivación, energía y posibles molestias o lesiones.
Generación automática de rutinas de entrenamiento personalizadas mediante IA.
Visualización de rutinas y seguimiento del progreso.
Integración con bases de datos y servicios cloud para almacenamiento seguro.


Estado del proyecto
